ScutSDK
=================


ScutSDK简介
----------------
Scut游戏服务器引擎开发的宗旨是为了能让网游开发和单机一样简单，以便于降低有志于游戏开发的小伙伴们进入网络游戏开发的门槛。ScutSDK是和Scut游戏服务器引擎，简化客户端开发的配套SDK，她彻底打通了Scut开源游戏服务器引擎与客户端引擎（如Cocos2d-x/Quick-x/Unity3D）项目间的通信，进而实现整套的网络游戏解决方案。
